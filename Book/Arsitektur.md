# Project Toko Buku Pelajar


| Layer      | Teknologi              |
| ---------- | ---------------------- |
| Frontend   | **Nuxt.js (Vue 3)**    |
| Backend    | **Laravel (hanya API)** |
| Auth       | Laravel Sanctum        |
| Database   | Sqlite                  |
| Arsitektur | REST API               |


```bash
Nuxt -> /http(Request)/ -> Laravel API / -> Sqlite app
```
